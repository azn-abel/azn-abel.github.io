Code Brown

Program Notes

Code Brown is an electronic piece composed in Logic Pro. It is Abel Lu's first ever composition. Code Brown uses kick, hi-hat, ride, and clap samples from Logic Pro's library and three custom samples made from flatulence, which provide the bass and other effects throughout the piece. Abel wanted to explore whether unconventional sounds - even ridiculous ones - could be made to work in a musical context. He believes the answer is yes.

Composer Bio

Abel Lu is an undergraduate student studying computer science at North Carolina State University. He has always been curious about how electronic music is made and enrolled in MUS 306 as a chance to find out. Code Brown is his first composition.
